# go-libs

Biblioteca de utilitários reutilizáveis para aplicações Go backend.

Cada lib é um módulo Go independente — importe somente o que precisar.

| Lib | Módulo | Descrição |
|-----|--------|-----------|
| [errors](./errors) | `github.com/LeoRBlume/go-libs/errors` | Mapeamento de domain errors para HTTP status codes |
| [logger](./logger) | `github.com/LeoRBlume/go-libs/logger` | Logging estruturado em JSON com rastreabilidade por contexto |
